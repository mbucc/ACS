ACS Documentation : ACS Core Architecture Guide : API Publication

The Big Picture

The ACS Core defines distinct publication methods for the three basic types of API that APM packages provide:
  1. Tcl Procedures: ad_proc and ad_library
  2. URLs: ad_page_contract
  3. Database views and stored procedures/functions
There is now an integrated user interface for browsing published APIs of all three types. Documentation for this system is available.
