Publishing Database Views and Stored Procedures/Functions

by Michael Yoon on 6 July 2000

ACS Documentation : ACS Core Architecture Guide : API Publication : Publishing Database Views and Stored Procedures/Functions

As of 3.4, ACS provides the /doc/view-pl-sql URL API (e.g., /doc/view-pl-sql?name=ad_general_permissions&type=package), which simply queries the Oracle data dictionary.

In the future, we plan to do more with schema browsing (see also the Schema Browser, which only presents tables, as of ACS 3.4).
