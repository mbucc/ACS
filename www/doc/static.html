Static File Support

part of the ArsDigita Community System by Philip Greenspun
  • User-accessible directory: /comments/, /links/
  • Site administrator directory: /admin/static/
  • data model : within /packages/acs-core/community-core.sql
  • Tcl procs: /tcl/ad-html; /tcl/ad-user-content-map;

The Big Idea

Is explained fairly thoroughly in Chapter 3 of Philip and Alex's Guide to Web Publishing.

The Medium-size Idea

Copy all the files on the server into an Oracle table. Give each one a unique ID. Collect comments and related links on each page. Display these in-line via the procedures in /tcl/ad-html

The filter in /tcl/ad-user-content-map records which users have examined which files.
