Patches for this installation

of the ArsDigita Community System
Any changes, enhancements or fixes that you make to the core ArsDigita software should be logged on this page. The information will then be useful (a) when upgrading this installation to a new version of the ACS (we'll know which files have to be customized), (b) when building new versions of the ACS core distribution, we can pull improvements and fixes from all of our installations merely by looking at /doc/patches on each one.
  • suggested format: filename; change description; date; programmer
  • e.g.: /bboard/q-and-a; made searching submit button appear even with non-MSIE browsers; 6/3/98; philg
